# Banking App

## What is it?

This is banking application where user can calculate and deposit and withdraw money and also can bankrupt.

## Technologies used

- React
- Redux
- typescript
- Styled-components

## How to use it?

- Go to the app
- Make a deposit or withdraw
- See your balance

## Features

- Deposit and withdraw money
- See your balance
- Bankruptcy
